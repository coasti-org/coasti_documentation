# Introduction

There are three steps to entering the coasti world:

1. Jump into the [coasti demo server](/docs/getting-started/demo-server)
2. Learn how to [install coasti](/docs/getting-started/install-coasti-content) and content packages on your own machine
3. Start [creating your own coasti content](/docs/getting-started/create-content)
